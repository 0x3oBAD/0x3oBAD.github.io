# Favicon Files

This directory contains the favicon files for your site.

## Current Files

- `favicon.svg` - Modern SVG favicon (scalable, works in all modern browsers)

## To Add More Formats (Optional but Recommended)

For maximum compatibility, you can also add:

1. **favicon.ico** - For older browsers (16x16 or 32x32 pixels)
   - Convert your SVG using: https://realfavicongenerator.net/ or https://favicon.io/
   
2. **apple-touch-icon.png** - For iOS devices (180x180 pixels)
   - Create from your SVG using any image editor

3. **android-chrome-192x192.png** and **android-chrome-512x512.png** - For Android
   - Create from your SVG using any image editor

## Quick Conversion Tools

- **Favicon Generator**: https://realfavicongenerator.net/
  - Upload your SVG and it will generate all formats automatically
  
- **Favicon.io**: https://favicon.io/
  - Can convert text/images to favicons

## Customizing the SVG

You can edit `favicon.svg` to change:
- Colors (currently dark background #0a0a0a, green text #00ff41)
- Text (currently "0x")
- Design (add shapes, symbols, etc.)

The Chirpy theme will automatically use these files - no configuration needed!
